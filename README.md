# 🌍 Community Connect - Hyperlocal Engagement Platform

## 📌 Project Overview
Community Connect is a platform that fosters neighborhood collaboration, resource-sharing, skill-sharing, and eco-consciousness.

## 🚀 Tech Stack
### Frontend:
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend:
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
